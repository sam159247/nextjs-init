# nextjs-init

Taking a note to remember how initial a [next.js](https://nextjs.org/) project.

## Specification

- Next.js typescript
- pnpm package manager tool
- husky
- prettier
- Airbnb JavaScript Style Guide

## Pre-install

```shell
brew install pnpm
```

## Usage

```bash
bash nextjs-init.sh {project_name}
```
